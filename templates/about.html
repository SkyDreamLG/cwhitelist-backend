关于 CWhitelist

CWhitelist 白名单管理系统

一个专为 Minecraft 服务器设计的现代化白名单管理系统。

主要特性：
  • 支持多种验证方式：玩家名称、UUID、IP地址
  • 完整的 API 接口，支持与 Minecraft 模组集成
  • 直观的网页管理界面
  • 支持多服务器管理
  • 详细的登录日志记录
  • 灵活的权限管理系统
系统信息：
  • 版本：1.0.0
  • 开发语言：Python (Flask)
  • 数据库：SQLite / MySQL
  • 前端框架：Bootstrap 5
注意事项

此系统为开发版本，请勿在生产环境中使用默认密码。
建议在部署前修改所有默认设置。
